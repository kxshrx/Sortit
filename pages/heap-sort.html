Heap Sort

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure. It works by first building a max-heap from the array, then repeatedly extracting the maximum element and rebuilding the heap until the array is sorted. This algorithm is notable for its consistent O(n log n) time complexity and in-place sorting capability.
