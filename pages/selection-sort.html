Selection Sort

Selection Sort is a simple sorting algorithm that works by repeatedly finding the minimum element from the unsorted portion of the array and placing it at the beginning. The algorithm maintains two subarrays: one that is already sorted and another that is unsorted. In every iteration, the minimum element from the unsorted subarray is picked and moved to the sorted subarray.

Complexity

Case Time Complexity Space Complexity
Best Case O(n²) O(1)
Worst Case O(n²) O(1)
Average Case O(n²) O(1)
